## 58. 最后一个单词长度（解）


### 思路
1. 通过两侧空格，
2. 根据空格拆分字符串成数组
3. 获取数组最后一个元素的长度

（不知道大佬们是怎么做到100%的）

```js
/**
 * @param {string} s
 * @return {number}
 */
var lengthOfLastWord = function(s) {
    let res = s.trim();
    res = res.split(" ");
    return res[res.length - 1].length
};
```
